mozart-chart
============

An javascript libary that make free and easy charts. Currently only can make Bar chart.

Option for chart:
- width
- height
- max
- min
- values (must look like see values)
- type (currently only "bar")

Values like this:

[
  key1: [1,2,3,4],
  key2: [4,3,2,1]
]
